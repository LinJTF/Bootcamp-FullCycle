# Comandos ``` cmd ``` utilizados durante o projeto

Para subir o docker compose: ``` docker-compose up -d ```

Para listar os containers ativos: ``` docker ps ```

Quando inserir uma biblioteca não nativa do go: ``` go mod tidy ```
